```
BRAND PROFILE: CUSTER
════════════════════════════════════════════════════════════════

🎨 PALETA VISUAL
Primario: Azul Royal #4F46E5 (profesionalismo, confianza)
Secundario: Amarillo Vibrante #FCD34D (urgencia, CTA)
Neutro: Blanco Puro #FFFFFF (contraste)
Acento: Rosa/Púrpura #C084FC (gradientes esquinas)
Oscuro: Azul #1E3A8A (tipografía en cajas)

📝 TIPOGRAFÍA
Headlines: Sans-serif Bold (Inter/Poppins), 45-55% espacio
- Color: Amarillo ó Blanco (alternancia)
Body: Regular (400), azul oscuro ó gris
- Tamaño: 60-70% más pequeño que headline
- Leading: 1.5-1.6x

📐 COMPOSICIÓN
Patrón: Z-diagonal (texto bold centro → CTA inferior derecha)
Espacio negativo: 35-40%
Gradientes decorativos: Rosa/purple en esquinas (sutil)

🎨 ELEMENTOS
Ilustraciones: 3D moderno, cartoonish, colorido
Botones: Redondeados (#FCD34D ó naranja), esquina inferior
Sin: Fotografía real corporativa

💬 TONO
Registro: Formal-conversacional (B2B pero amable)
Headlines: 5-12 palabras, PROVOCACIÓN + CONTRASTE
- Ej: "Una landing que vende NO EMPIEZA con diseño"
Subtítulo: Expande problema (8-15 palabras)
CTA: Verbal directo ("Conocé", "Accedé", no "Enviar")
Voz: Consultor experto + disruptor educador

════════════════════════════════════════════════════════════════
```

## 🔧 PROTOCOLO DE ANÁLISIS (Ejecutar en Orden)

### **═══ ANÁLISIS 1: CONVERSION ARCHITECT ═══**

Analiza como Estratega de Performance Marketing especializado en Meta Ads.

**Tareas:**
1. ¿El orden de las plaques maximiza conversión/consideración?
   - ¿Placa 1 es Hook visual (<3 segundos)?
   - ¿Flujo lógico: Problema → Impacto → Solución → Valor → Acción?

2. Propón reordenamiento si es necesario

3. Para CADA PLACA define:
   - Hook Visual: Qué debe destacar
   - Distribución: Titular 40% | Subtítulo 35% | Espacio 25%
   - Color estratégico: Cuál usar + psicología

4. Especifica:
   - ¿Diferencia feed (cuadrado) vs Stories (vertical)?
   - ¿Cómo adaptar si es ambos?

**Output esperado:**
```
✓ Orden actual: [LANZABLE / REORDENAR]
✓ Hook Placa 1: [Análisis]
✓ Flujo: Problema → Impacto → Solución → Valor → Acción ✓
✓ Recomendaciones por formato
```

### **═══ ANÁLISIS 3: BRAND GUARDIAN ═══**

Actúa como Director de Arte validador.

**Tareas:**
1. ¿Cumple el Brand Profile de Custer?
   - ✓ Paleta (Azul royal, amarillo, blanco, rosa en esquinas)
   - ✓ Tipografía (Bold headlines, aéreo, alternancia color)
   - ✓ Composición (Z-diagonal, 35-40% espacio negativo)
   - ✓ Elementos (3D moderno, botones redondeados amarillos)
   - ✓ Tono (Provocador, problema-solución, 5-12 palabras)

2. ¿Coherencia tonal en TODAS las plaques?
   - ¿Todas transmiten el mensaje central?
   - ¿Hay inconsistencia tipográfica?
   - ¿Copy mantiene tono "consultor disruptor"?

3. ¿Progresión lógica?
   - ¿Las plaques avanzan hacia el objetivo?
   - ¿Cierre propone acción clara?

4. ¿Accesibilidad visual?
   - Contraste título vs fondo: ≥4.5:1
   - Legibilidad mobile: Texto no ocupa >50% espacio

**Output:**
```
✓ CUMPLE BRAND PROFILE: [SÍ/PARCIAL/NO]

Desvíos encontrados:
- [Desvío 1 + corrección específica]
- [Desvío 2 + corrección específica]

Coherencia tonal: [✓ Consistente / ⚠️ Mejoras necesarias]

Accesibilidad: [✓ Aprueba / ⚠️ Ajustar contraste]
```

### **═══ ANÁLISIS 5: CRITICAL AUDITOR ═══**

Realiza el "Squint Test" (entrecerrar ojos).

**Tareas:**

1. **Punto Focal por Placa:**
   - ¿Qué destaca en <100ms con ojos entrecerrados?
   - ¿Compite headline con otros elementos?
   - ¿Hay claridad absoluta?

2. **Gestalt (Carrusel como CONJUNTO):**
   - ¿Se siente como UNA campaña Custer ó 10 ads sueltos?
   - ¿Hay consistencia visual entre plaques?
   - ¿Elementos se sienten "de la familia"?

3. **Carga Cognitiva:**
   - Placa 1: ¿Se entiende el hook en <2 segundos?
   - Placa 2-4: ¿La continuidad es clara?
   - Placa 5 (Cierre): ¿Acción sin ambigüedad?

**Output:**
```
🎯 VEREDICTO: [✅ LANZABLE / ⚠️ AJUSTES MENORES / ❌ NECESITA REVISIÓN]

Punto Focal:
- Placa 1: ✓ [Elemento dominante]
- Placa 2: ✓ [...]
- [etc]

Cohesión Visual:
- Sensación de marca: ✓ Fuerte [por qué]
- Consistencia: ✓ [Análisis]

Cambios Prioritarios (máximo 3):
1. [Cambio 1] - Impacto: Alto
2. [Cambio 2] - Impacto: Medio
3. [Cambio 3] - Impacto: Bajo

Listo para: [Diseño / Revisión interna / Publicación]
```
