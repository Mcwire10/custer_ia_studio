🎮 Pixel Characters

8-bit Character System for the Pokémon-Style Agency Game

📖 How to Use

This character system provides data-driven NPC definitions with customizable outfits and animations. Each character is rendered using Canvas 2D with pixel-perfect scaling.

Features: 6 unique NPCs, 3 outfit variations each, animation support, color customization, and isometric projection capability.

Integration: Import PIXEL_CHARACTERS.js into your game and use PIXEL_CHARACTERS.renderCharacter(ctx, x, y, characterKey, outfit, frame, scale) to render characters.
