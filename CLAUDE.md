## 🎨 LOS 5 MCPs PARA DISEÑO & MARKETING (NUEVOS)

**¿Qué hace?**
- Claude accede a tus archivos de Figma
- Modifica diseños automáticamente
- Crea componentes nuevos

**Ejemplo:**
```
Tú: "Cambiar el color del botón a azul en Figma"
Claude: (Accede a Figma) → Cambia automáticamente el color
```

**Lo usas para:** Interfaces profesionales, prototipos, componentes reutilizables

## 🚀 Cómo Se Usan Juntos

**Flujo Completo de Diseño + Marketing:**

```

   ↓

   ↓

   ↓
4. Google Workspace MCP (crea email con el diseño)
   ↓

   ↓

   ↓
Resultado: Campaña COMPLETA en minutos
```

## 🤖 Para Agentes Automáticos

### Task Master MCP Integration

Si necesitas gestión automática de tareas:

```json
{
  "mcpServers": {
    "task-master-ai": {
      "command": "npx",
      "args": ["-y", "task-master-ai"],
      "env": {
        "TASK_MASTER_TOOLS": "standard",
        "ANTHROPIC_API_KEY": "${process.env.ANTHROPIC_API_KEY}"
      }
    }
  }
}
```

Guardar en `.mcp.json` en la raíz del proyecto.

### Invocar Skills Automáticamente

Los agentes pueden detectar automáticamente cuándo usar un skill:

```javascript
// Si necesitas copywriting → /copywriting
// Si necesitas SEO → /seo-audit
// Si necesitas PDF → /pdf
// etc.
```
