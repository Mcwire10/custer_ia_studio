HOW TO USE
================================================================================

1. COPY FILES:
   - Copy PIXEL_CHARACTERS.js to your project
   - Copy CHARACTER_SYSTEM_EXAMPLES.js (optional)

2. INCLUDE IN HTML:
   <script src="./PIXEL_CHARACTERS.js"></script>

3. BASIC USAGE:
   const ctx = canvas.getContext('2d');
   ctx.imageSmoothingEnabled = false;
   PIXEL_CHARACTERS.renderCharacter(ctx, 100, 100, 'juan', 0, 0, 2);

4. WITH ANIMATION:
   const animator = new PIXEL_CHARACTERS.AnimationController('sofia', 0);
   animator.startWalking();
   animator.update();
   const frame = animator.getCurrentFrame();

5. USING EXAMPLES:
   const room = new CHARACTER_EXAMPLES.RoomDisplay(canvas, [
     { key: 'juan', outfit: 0, x: 100, y: 150 },
     { key: 'sofia', outfit: 1, x: 300, y: 150 }
   ]);
   room.render();

================================================================================
TROUBLESHOOTING
================================================================================

Q: Characters not showing?
A: Ensure ctx.imageSmoothingEnabled = false, use lowercase keys

Q: Animation not working?
A: Call animator.update() every frame in game loop

Q: Characters pixelated appearance lost?
A: Set ctx.imageSmoothingEnabled = false

Q: Performance issues with multiple NPCs?
A: Cache sprite sheets, limit animations, use appropriate scaling

For more help: See PIXEL_CHARACTERS_GUIDE.md troubleshooting section
